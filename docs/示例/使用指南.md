# docsify 使用指南

## 添加新文章

直接把 `.md` 文件扔到 `docs/` 对应文件夹下，然后更新 `_sidebar.md` 加上导航。

## 目录规范

```
docs/
├── 示例/          ← 示例文件夹
│   └── 使用指南.md
├── 2026-05-27/    ← 按日期组织，每天一个文件夹
│   ├── 文章1.md
│   └── 文章2.md
```

## 侧边栏格式

```markdown
* [首页](/)
* 文章分组名称
  * [文章标题](docs/xxx/文章.md)
```

## 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. push 这个文件夹的内容
3. Settings → Pages → Source 选 `main` 分支，根目录 `/`
4. 等几秒钟就上线了
