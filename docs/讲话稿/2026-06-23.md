# 在全市数字经济与实体经济深度融合调研座谈会上的讲话

**（2026年6月23日）**

同志们：

今天我们召开全市数字经济与实体经济深度融合调研座谈会，主要目的是深入贯彻党中央关于加快数字经济发展、推动数实融合的战略部署，全面了解我市数实融合推进情况，找准瓶颈问题，研究突破路径，为下半年工作定向把舵。会前，调研组用两周时间走访了8个县区、32家重点企业、6个产业园区，形成了详实的调研报告。刚才，几位同志的发言既有数据支撑，也有实践感悟，提出了很多有价值的建议。下面，我结合调研情况，讲三点意见。

## 一、充分肯定成绩，清醒认识差距

数实融合是我市转型升级的关键抓手。今年以来，全市上下围绕"数字赋能、融合创新"主线，取得了一定成效：

一是数字基础设施持续夯实。全市5G基站累计达到8760座，覆盖率达94.6%，千兆光纤入户率突破72.3%。工业互联网标识解析二级节点上线运行，注册企业628家，解析量突破1.2亿次。沈阳数字经济产业园二期竣工，新增算力中心2座、总算力达到380PFLOPS。

二是企业数字化转型稳步推进。实施"智改数转"三年行动以来，累计推动412家规上企业完成数字化改造，改造覆盖率达到46.8%，其中智能工厂16家、数字化车间52个。装备制造、医药化工、食品加工三大主导行业数字化率分别达到52.1%、38.7%和31.5%，较去年同期分别提升6.3、4.8和3.2个百分点。

三是数字产业规模稳步增长。数字经济核心产业产值达到862亿元，同比增长14.6%，占GDP比重提升至14.3%。软件和信息服务业收入215亿元，增长16.2%；大数据和云计算产业收入48亿元，增长22.7%。新增数字经济领域市场主体2800户，增长18.9%。

四是融合应用场景不断丰富。打造了27个数实融合典型场景，涵盖智能制造、智慧农业、数字商贸、智慧医疗等领域。"沈阳智造"云平台注册企业1.8万家，撮合交易额62亿元。农业数字化试点覆盖面积28万亩，农产品电商销售额增长31.4%。

但必须清醒看到，我市数实融合仍存在明显差距和突出问题：

**融合深度不够。**412家改造企业中，达到L4级（深度集成）的仅占8.7%，L3级（系统协同）的占22.6%，超过半数仍停留在L1-L2（局部替代、单项应用）阶段，"改设备不改流程、上系统不上数据"现象普遍。

**中小企业转型滞后。**全市2.1万家中小制造企业中，开展数字化改造的不足12%，远低于长三角26%的平均水平。成本高、人才缺、不敢试、不会转，是制约中小企业转型的四大痛点。

**数据要素活力不足。**政务数据开放率仅为38.2%，低于全国平均51.7%的水平。企业间数据共享机制缺失，工业数据流通量仅占全国的1.8%。数据要素市场建设尚处于起步阶段，尚未形成定价、交易、确权的完整链条。

**数字人才供给缺口大。**全市数字经济领域人才缺口约6800人，其中大数据、人工智能、工业互联网等核心方向缺口占比62.3%。本地高校年输送相关专业毕业生仅4200人，留存率不足35%，"引不来、留不住"问题突出。

## 二、聚焦关键突破，打好数实融合攻坚战

下半年，要围绕"深度融合、全面赋能"目标，重点打好三场攻坚战：

**（一）打好企业深度转型攻坚战**

一是实施"千企智改"提速计划。下半年再推动200家规上企业完成数字化改造，力争年底改造覆盖率突破55%，L3及以上占比提升至35%以上。对率先达到L4级的企业，给予一次性奖励50万元。

二是破解中小企业"不敢转、不会转"难题。推出"小快轻准"数字化解决方案包，针对食品加工、装备制造、医药化工三个行业分别开发3套标准化方案，单套成本控制在5万元以内，政府补贴50%。建设3个中小企业数字化转型服务中心，提供诊断评估、方案匹配、实施监理全流程服务，下半年服务企业不少于800家。

三是强化标杆引领。遴选8家数字化转型标杆企业，组建"沈阳智改数转联盟"，开展"标杆带百企"结对帮扶活动，每家标杆企业结对帮扶12-15家中小企业，年底帮扶覆盖率达到30%以上。

**（二）打好数据要素激活攻坚战**

一是加快政务数据开放。制定《政务数据开放目录（第三批）》，新增开放数据集120个，年底政务数据开放率提升至55%以上。重点开放市场监管、公共信用、产业政策等企业高频需求数据，做到"应开放尽开放"。

二是建设工业数据流通平台。依托工业互联网标识解析体系，搭建市级工业数据共享交换平台，年底接入企业300家以上，数据流通量提升3倍。探索数据资产入表、数据信托等创新机制，选取5家企业开展试点。

三是培育数据要素市场。制定《数据要素市场化配置改革方案》，建立数据确权登记、定价评估、交易流通基础制度框架。推动沈阳大数据交易中心挂牌运营，年底实现交易额突破5000万元。

**（三）打好数字人才引育攻坚战**

一是实施"数智人才领航计划"。下半年引进数字经济高层次人才350人以上，其中大数据、人工智能、工业互联网方向占比不低于60%。对引进的博士及以上人才，给予安家补贴30万元、科研启动经费20万元。

二是深化产教融合育才。推动8家重点企业与东北大学、沈阳工业大学等6所高校共建"数实融合产业学院"，开设智能制造、工业互联网、数据科学3个交叉专业方向，下半年招生800人。建设4个数字经济实训基地，年培训在职人员3000人次。

三是营造留才生态。完善数字经济人才服务数字化平台功能，实现人才政策"一键匹配、免申即享"。下半年人才公寓新增投放500套，数字经济领域人才留存率提升至45%以上。

## 三、强化机制保障，确保融合攻坚落地见效

**强化组织推进机制。**成立市数实融合工作推进领导小组，由分管副市长担任组长，工信、发改、科技、数据等部门主要负责同志为成员，每月召开调度会，季度通报进展。各县区同步建立推进专班，形成市县联动、条块协同的工作格局。

**强化政策资金保障。**整合现有数字经济专项资金1.2亿元，新增安排数实融合专项8000万元，重点用于中小企业转型补贴、数据要素市场建设、人才引育激励。设立数实融合产业基金，首期规模5亿元，引导社会资本参与。

**强化考核督导机制。**将数实融合关键指标纳入各县区、各部门年度考核体系，设定7项量化指标：规上企业改造覆盖率、中小企业改造率、政务数据开放率、数字经济核心产业占比、数据流通量、数字人才新增数、融合场景新增数。实行月调度、季评估、半年考核，对进展滞后单位下发预警通知，连续两季度排名末位的由领导小组约谈主要负责同志。

同志们，数字经济与实体经济深度融合是时代大势、发展必然，也是沈阳转型升级、振兴突破的关键一招。当前，我们既面临深度融合的攻坚期，也迎来乘势而上的窗口期。各级各部门要以"坐不住"的紧迫感、"等不起"的责任感、"慢不得"的危机感，主动作为、攻坚克难，让数字真正赋能产业、让数据真正创造价值、让数实融合真正成为沈阳高质量发展的强大引擎！
