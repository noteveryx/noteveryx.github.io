# 在全市基层治理现代化调研座谈会上的讲话

**（2026年5月25日）**

同志们：

今天，我们专程来到基层一线，召开这次调研座谈会，目的是深入了解基层治理的真实情况，听听大家在实际工作中遇到的困难和问题，共同研究推进基层治理现代化的思路和举措。大家都是长期奋战在基层一线的同志，最有发言权、最懂实情。我希望大家打开天窗说亮话，把真实想法、真实困难、真实诉求讲出来，我们一起把问题搞清楚、把对策定实在。

前一段时间，市委市政府对基层治理工作高度重视，各区县、各部门做了大量卓有成效的工作。今年1至4月，全市社区（村）信访事项化解率达到91.3%，同比提升6.8个百分点；基层矛盾纠纷调处成功率达88.7%，网格化服务管理覆盖率达到100%。这些成绩来之不易，凝结着在座各位和广大基层干部的辛勤付出。在此，我代表市委市政府，向大家表示衷心的感谢和崇高的敬意！

但也要清醒地看到，当前基层治理仍面临不少深层次矛盾和问题。有的地方人员力量不足、权责不匹配；有的地方数字化手段运用不充分，数据"孤岛"现象依然存在；有的地方干部激励机制不完善，基层干部疲于应付、干劲不足。这些问题，需要我们下大力气、动真格地加以解决。

下面，我讲三个方面的意见。

## 一、切实摸清底数，把基层治理的"痛点""堵点""难点"找准找实

调查研究是谋事之基、成事之道。这次座谈，我们就是要把功夫下到摸底数上，把问题找准找实，不搞形式、不走过场。

**一是找准资源与需求的错位问题。** 当前，部分社区（村）反映"多个系统、多个平台、多个口子报数据"，基层干部平均每周花在填报数据上的时间超过12小时，有效服务群众的时间被大量挤占。我们要在这次调研中，把这类"表格政治""材料政治"问题的根源找清楚，拿出切实的精减措施。

**二是找准职责与权力的不对等问题。** 基层治理"上面千条线、下面一根针"，社区（村）承接了大量行政事务，但相应的执法权、处置权、资金配置权却十分有限。要厘清哪些权力可以依法依规下放，哪些职责需要重新分配，构建权责对等的基层治理体系。

**三是找准服务与群众需求的差距。** 通过走访了解，目前全市60岁以上老年人中，仍有约18.4%未能享受到便捷的数字化养老服务；残疾人、低保边缘家庭等特殊群体的个性化服务需求满足率仅为74.2%，与群众期盼还有明显差距。要聚焦重点群体，把服务的"最后一公里"打通打实。

## 二、着力补齐短板，推动基层治理能力整体提升

基层强则全局稳，基层弱则全局堪忧。补齐基层治理短板，是推进治理现代化的关键所在。

**一要强化人才支撑。** 当前全市社区专职工作者平均年龄为42.6岁，其中大专及以上学历占比仅为61.3%，专业化水平有待提升。今年要重点推进"百名选调生驻社区"计划，同步深化"村干部学历提升工程"，力争到2026年底，社区（村）书记中大专及以上学历占比提升至85%以上。要建立健全基层干部容错纠错机制，让敢于担当的干部在基层干出一番事业。

**二要深化数字赋能。** 加快推进"一网统管"平台建设，整合现有17个分散系统，年内完成8个高频应用场景的集成并轨。以数字技术为基层减负，让"数据多跑路、干部少跑腿"。力争到年底，基层干部日常行政事务处理时间压缩30%以上，腾出更多精力服务群众。同步推进"智慧网格"建设，在全市2763个网格中配备移动端智能终端，实现网格事件处置全流程可追溯、可评价。

**三要完善激励机制。** 今年要在全市范围内落实基层干部"三个倾斜"政策——职务晋升向基层一线倾斜、绩效奖励向艰苦岗位倾斜、荣誉表彰向实绩突出倾斜。切实解决"基层苦、基层累、待遇低"的问题，让基层干部有盼头、有奔头、有干头。探索建立基层治理积分制度，将居民参与社区事务的情况纳入积分管理，通过兑换礼品、优先享受公共服务等方式，形成"人人参与治理"的良好格局。

## 三、聚焦共建共治，构建多元参与的基层治理新格局

基层治理不是政府一家的事，必须充分激活社会活力，走好共建共治共享之路。

**一要发挥党建引领核心作用。** 充分发挥基层党组织战斗堡垒和党员先锋模范作用，推动"双报到"机制走深走实。今年要在全市1247个社区中实现"在职党员进社区报到"覆盖率达到100%，年均参与社区公益活动不少于3次。对基层党建工作较弱的社区实行重点帮扶，确保每个薄弱社区有1名处级以上干部联系包保，年内完成全面整改提升。

**二要用好社会组织和专业力量。** 当前全市在册社会组织3142家，但真正参与基层治理、提供专业服务的比例不足40%。要推动"社会组织孵化工程"落地，通过政府购买服务、项目资助等方式，重点培育一批专业化的社区服务类、矛盾调处类、老年关怀类社会组织。到2026年底，全市参与基层治理的专业社会组织数量要突破800家，直接服务居民人次突破120万。

**三要构建共建共治合力。** 深化"街道吹哨、部门报到"机制，打通部门协作"肠梗阻"。推广"社区议事厅"模式，将涉及居民切身利益的重大事项纳入协商议事范围，确保群众的事由群众来议、由群众来决。建立基层治理考核评价体系，将"群众满意度"作为硬指标，纳入各区县党委政府年度目标考核，权重不低于20%，以考核导向促进治理实效。

同志们，基层是国家治理体系的"神经末梢"，也是服务群众的"最前哨"。推进基层治理现代化，是一项长期的系统工程，不可能一蹴而就，但也绝不能等待观望。每一位在座的同志，都是这项事业的重要推动者。希望大家带着问题去调研、带着责任去落实、带着感情去服务，把群众的每一件小事都当作大事来办，把"最后一公里"的每一步都走得更稳更实。

市委、市政府将全力做好服务保障，在政策支持、资金投入、力量配备等方面加大力度，切实为基层减负增效。我们有信心、有决心，在全市上下的共同努力下，走出一条具有本地特色的基层治理现代化之路，让人民群众在家门口感受到更多获得感、幸福感、安全感！

谢谢大家！
