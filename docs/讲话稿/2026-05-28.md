# 在全市人才工作高质量发展推进会上的讲话

**（2026年5月28日）**

同志们：

今天我们召开全市人才工作高质量发展推进会，主要任务是深入学习贯彻习近平总书记关于做好新时代人才工作的重要思想，全面落实省委、市委人才工作部署要求，总结前一阶段工作成效，分析当前面临形势，对下一阶段人才工作进行再动员、再部署、再推进。

今年以来，全市上下坚持"人才是第一资源"理念不动摇，以更大力度推进人才强市战略，各项工作取得了阶段性成效。一是引才规模持续扩大，上半年累计引进各类高层次人才1200余人，其中博士学历以上人才同比增长18%，引进数量创近五年同期新高。二是平台载体不断夯实，人才服务数字化平台上线运行以来，累计服务人才超8万人次，办理事项3.2万件，人才办事平均时长压缩至2.8个工作日，较改革前缩短42%。三是政策体系逐步完善，先后出台《高层次人才安居保障实施办法》《青年人才创业补贴实施细则》等配套政策12项，形成了较为完备的"1+N"人才政策矩阵。

在肯定成绩的同时，我们必须清醒地看到工作中存在的短板和不足：高端领军人才仍然偏少，院士级顶尖人才仅有3人驻站；产才融合不够紧密，本地高校毕业生留沈率不足45%；区域竞争日趋激烈，周边城市"抢人"力度持续加大，我们的紧迫感必须进一步增强。

## 一、聚焦精准引育，打造人才集聚"强磁场"

人才引进要由"大水漫灌"向"精准滴灌"转变，做到有的放矢、靶向发力。

**一是在重点领域精准发力。** 围绕先进制造业、数字经济、生物医药、新材料等主导产业需求，编制发布2026年度紧缺人才目录，锁定100个关键岗位、500名急需人才。深化"带土移植"工程，力争全年引进高水平创新创业团队15个以上，每个团队给予最高1000万元的综合扶持。

**二是在青年群体重点突破。** 实施万名青年英才集聚计划，面向"双一流"高校开展校园招聘专场活动不少于50场，全年吸纳本科及以上应届毕业生不低于1.5万人。用好"沈阳兴沈英才计划"，将青年人才购房补贴上限提高至8万元，租房补贴覆盖范围扩大至毕业五年内全部本科生。

**三是在柔性用才创新模式。** 不求所有、但求所用，推广"候鸟专家""周末工程师"等柔性引才模式，与国内20所一流高校建立常态化合作关系，设立离岸研发中心或人才工作站不少于10个，柔性引进高层次人才300人以上。

## 二、聚焦平台赋能，构筑干事创业"主阵地"

平台是人才施展才华的舞台，要把平台建得更优、更强，让人才来了有舞台、干得好能出彩。

**一是加快数字赋能提效。** 全力推进人才服务数字化平台二期建设，年底前实现人才认定、项目申报、待遇兑现等核心业务"一网通办""全程网办"。推动数据跨部门共享打通，将人社、教育、住建、公安等8个部门的人才相关数据纳入统一平台，实现"一次采集、多方共用"，让数据多跑路、人才少跑腿。目标是平台注册用户突破15万，网上办结率达到95%以上。

**二是强化载体梯次培育。** 支持现有3家国家级博士后科研工作站、12家省级工作站提质增效，新申报省级以上研发平台5家。高标准建设人才公寓2000套以上，打造15分钟人才服务圈，在重点产业园区配套建设人才驿站、创业孵化器等基础设施。

**三是深化校地企合作联动。** 建立政校企三方定期对接机制，每季度召开一次产才融合对接会，推动驻沈高校优势学科与本地龙头企业结对共建。全年新增产学研合作项目80个以上，促成科技成果本地转化交易额突破10亿元，让更多人才在沈阳既能搞科研又能出成果。

## 三、聚焦生态优化，厚植近悦远来"沃土壤"

环境好则人才聚，人才聚则事业兴。要以最大诚意、最优服务让人才安心安业。

**一是完善全周期服务体系。** 推行"人才服务专员"制度，为每位高层次人才配备一名服务专员，提供从落户、住房、子女入学到医疗保障的"一对一"全程帮办。开通人才服务热线和线上咨询通道，确保诉求响应率100%、办结满意率98%以上。

**二是健全多元化激励机制。** 修订完善人才评价机制，坚决破除"唯论文、唯职称、唯学历、唯奖项"倾向，建立以创新价值、能力、贡献为导向的人才评价体系。加大对作出突出贡献人才的表彰奖励力度，评选表彰一批"兴沈杰出人才""优秀青年科技人才"，营造尊重劳动、尊重知识、尊重人才、尊重创造的浓厚氛围。

**三是强化资金投入保障。** 年度人才发展专项资金投入不低于上年水平，确保各项人才政策应兑尽兑、应落尽落。同时撬动社会资本参与人才工作，探索设立人才创投基金，引导金融机构推出"人才贷""人才保"等特色金融产品，为人才创新创业提供更有力的金融支撑。

同志们，人才竞争归根结底是发展环境的竞争、服务效能的竞争。全市各级各部门要牢固树立"一盘棋"思想，把人才工作摆在更加突出的位置，主要负责同志亲自抓、分管负责同志具体抓，形成齐抓共管、协同推进的工作格局。组织部门要发挥牵头抓总作用，各成员单位要各司其职、密切配合，以钉钉子精神抓好各项任务落实，奋力开创我市人才工作高质量发展新局面，为全面振兴全方位振兴提供坚强人才保证和智力支持！
