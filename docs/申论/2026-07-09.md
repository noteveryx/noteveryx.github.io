# 申论每日一练 — 2026年7月9日

## 给定材料

某省H市地处江淮分水岭，是传统农业大市，全市常住人口320万，其中农村人口占比近六成。2024年，全市村均集体经济收入仅为38万元，低于全省平均水平41万元，且呈现明显的"两极分化"——年收入百万元以上的"明星村"占比不足8%，而年收入10万元以下的"薄弱村"占比高达63%。该市A村曾是远近闻名的先进村，靠流转土地发展蔬菜产业，村民人均年收入突破2.6万元；但近年来，由于冷链物流跟不上、批发市场萎缩、销售渠道单一，村里的西红柿、黄瓜等时令蔬菜频繁出现"卖难"现象，部分大棚一度撂荒。村民老李算了一笔账："一亩大棚一年投入七八千，遇到滞销年份，能回本就不错了。"村干部也坦言，"村里的能人大多出去打工了，留下来的老年人种不动地，年轻人不愿种地的问题越来越突出。"

与此同时，H市农业产业结构存在三重矛盾：一是产业链条短，初级农产品占比超过70%，精深加工率不足15%；二是品牌效应弱，全市仅有12个地理标志农产品，但市场知名度普遍不高，在长三角商超中销售占比不足3%；三是人才支撑弱，全市农村实用人才仅占总人口的1.2%，基层农技推广队伍"老龄化""断层化"问题突出，45岁以上人员占比超过60%。

面对困局，H市2025年启动"千村示范、万村提升"工程，提出用三年时间全面消除年收入20万元以下的薄弱村，探索出"党支部+合作社+龙头企业+农户"的"四联"发展模式。但在推进过程中也暴露出新问题：部分地方盲目上马田园综合体项目，将"产业兴旺"简单理解为"村村点火、户户冒烟"；部分合作社与农户利益联结松散，沦为"老板村""空壳社"；个别地方片面追求村集体经济收入数字，搞"数字注水""账上富贵"，老百姓获得感并不强。

业内人士指出，发展新型农村集体经济，是推进乡村全面振兴的关键支撑，但绝不是简单"翻烧饼"或"造盆景"。只有真正尊重农民意愿、立足资源禀赋、健全利益联结，才能让集体经济在希望的田野上结出累累硕果。

## 作答要求

1. 根据给定材料，概括H市农村集体经济发展面临的主要问题，并提出相应的对策建议。（不超过400字）
2. 结合材料，自选角度，自拟题目，写一篇议论文。（800-1000字）

## 参考答案

### 第一题：概括问题与对策建议

**主要问题：**

1. 产业发展层次低。村集体经济以传统种植为主，初级农产品占比超70%，精深加工率不足15%，产业链条短、附加值低。

2. 集体经济两极分化。收入百万元以上的"明星村"不足8%，年收入10万元以下的"薄弱村"占比高达63%，区域差距明显。

3. 销售渠道与配套滞后。冷链物流跟不上、批发市场萎缩、电商等新渠道拓展不足，产销衔接不畅，"卖难"问题时有发生。

4. 人才支撑严重不足。农村实用人才仅占总人口1.2%，45岁以上基层农技人员占比超60%，"谁来种地""谁来经营"问题突出。

5. 经营主体带动力弱。合作社与农户利益联结松散，盲目上项目、"空壳社"等现象并存。

6. 考核导向出现偏差。部分地方片面追求数字，搞"账上富贵"，忽视农民真实获得感。

**对策建议：**

1. 立足资源禀赋，做好"土特产"文章。深挖地方特色，发展精深加工，打造区域公用品牌，提升产品附加值和市场竞争力。

2. 健全产销衔接体系。补齐冷链物流短板，畅通电商、直播、订单农业等多元销售渠道，让"好产品卖上好价钱"。

3. 强化人才支撑。实施"雁归工程""农技特岗计划"，鼓励大学生村官、返乡创业者扎根乡村，同时加强本土农民技能培训。

4. 完善利益联结机制。规范推广"党支部+合作社+龙头企业+农户"模式，让小农户深度嵌入产业链、分享增值收益。

5. 优化考核导向。摒弃"数字注水"，把农民满意度、增收实效作为考核硬指标，让集体经济回归"为民"初心。

### 第二题：议论文

**让集体经济在田野里结出"真果实"**

乡村振兴，产业为基；产业兴旺，集体经济为要。H市的实践证明，新型农村集体经济不是"翻烧饼"造"盆景"，唯有立足实际、尊重农民、健全联结，才能让广袤乡村真正旺起来、强起来。

**一要立足"土特产"，做好特色文章。** 乡村产业最忌"千村一面"。有的地方盲目跟风，看到别处搞大棚就跟风种蔬菜，看到别处搞民宿就一哄而上搞民宿，结果"水土不服"，产能过剩。集体经济要"活"，必须扎根脚下土地，依托资源禀赋，宜种则种、宜养则养、宜游则游，把"土"的资源做成"特"的文章，让"特"的产品变成"优"的品牌。唯有如此，集体经济才有根、有魂、有竞争力。

**二要紧扣"产业链"，补齐配套短板。** 不少村子不是没有产业，而是产业"链"上"断点"太多——生产环节不错，冷链物流跟不上；品质很好，但销售渠道单一；增产容易，增收难。要把集体经济做大做强，必须拉长产业链，从初级产品向精深加工延伸；健全供应链，补齐仓储、冷链、物流等基础设施；拓展销售链，搭上电商、直播、订单农业等新业态的快车，让优质农产品既产得好又卖得好。

**三要锚定"人才兴"，破解"谁来种地"困局。** 人是生产力中最活跃的因素。集体经济发展最缺的不是资金、不是项目，而是"能人"。要善于"筑巢引凤"，通过返乡创业支持、农技特岗、税收减免等政策，把有见识、有能力、有情怀的年轻人引回来；要注重"固本培元"，把传统农民培养成懂技术、会经营、善管理的新型职业农民，夯实集体经济的人才根基。

**四要健全"利益链"，做实联农带农。** 集体经济姓"公"，必须让农民真正受益。要规范合作社运营，防止"老板村""空壳社"现象，杜绝"大户通吃、小户靠边"；要完善利益分配机制，让农户以土地、资金、劳动力入股，参与产业链各环节，分享增值收益，让"村集体强"与"村民富"同频共振。

**五要树立"实导向"，纠正"数字注水"。** 考核是指挥棒。如果一味以账面收入论英雄，就会催生"账上富贵"的虚假繁荣。要把农民满意度、增收实效、利益联结紧密度等指标作为硬考核，让集体经济发展回到为民富民的根本立场上。

习近平总书记反复强调，"农业农村工作，说一千、道一万，增加农民收入是关键。"发展新型农村集体经济，归根结底是要让农民的钱袋子鼓起来、笑脸多起来。各级党委政府应科学谋划、久久为功，让集体经济在希望的田野上结出"真金白银"的果实，绘就"农业强、农村美、农民富"的崭新画卷。
