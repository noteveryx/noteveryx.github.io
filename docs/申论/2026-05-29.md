# 申论每日一练 — 2026年5月29日

## 给定材料

材料一：N市地处长江中下游，是传统工业重镇。过去十年间，GDP年均增速保持在8%以上，但付出了沉重的环境代价。2023年，全市空气质量优良天数仅占62%，主要河流Ⅳ类以上水质断面占41%，工业固废综合利用率不足50%。2024年初，N市启动"绿色转型三年行动"，关停搬迁了32家高耗能企业，但短期内出现了近万名工人转岗安置问题。部分企业主反映"环保标准一年比一年高，技改投入跟不上"，也有居民抱怨"工厂关了，空气好了，可饭碗没了"。

材料二：L县位于西部山区，曾是国家级贫困县。2019年脱贫摘帽后，依托丰富的林业资源和优美自然风光，大力发展生态旅游和林下经济。截至2025年底，全县生态旅游年接待游客超200万人次，林下种植面积达30万亩，农民人均可支配收入较脱贫时翻了一番。但快速发展的背后也存在隐忧：部分景区过度开发导致植被破坏，旺季游客量远超环境承载力，部分林下种植项目因缺乏科学规划出现水土流失苗头。

材料三：习近平总书记在2023年全国生态环境保护大会上强调，"绿水青山就是金山银山"，要协同推进降碳、减污、扩绿、增长，建设人与自然和谐共生的现代化。2024年国务院印发《关于加快经济社会发展全面绿色转型的意见》，明确提出到2030年重点领域绿色转型取得积极进展，到2035年绿色低碳循环发展经济体系基本建立。

## 作答要求

1. 根据给定材料，概括N市和L县在绿色发展中面临的主要问题，并提出对策建议。（不超过400字）
2. 结合材料，自选角度，自拟题目，写一篇议论文。（800-1000字）

## 参考答案

### 第一题：概括问题与对策建议

**主要问题：**

1. 传统工业城市转型阵痛突出。N市"先污染后治理"模式导致环境欠账多，关停搬迁企业在短期内造成就业压力，形成环保与民生的两难困境。
2. 企业环保技改能力不足。环保标准持续提高，中小企业资金和技术储备有限，技改投入跟不上政策要求。
3. 生态产业发展存在短视行为。L县生态旅游过度开发，旺季超载运营，缺乏环境承载力评估和游客分流机制。
4. 林下经济规划不科学。部分林下种植项目缺乏前期调研和环境影响评价，引发水土流失等次生生态问题。

**对策建议：**

一是建立产业转型缓冲机制。设立绿色转型专项基金，为关停企业职工提供技能培训和转岗补贴，引导企业向绿色产业链延伸，实现"关停不失业、转型有出路"。

二是加大中小企业技改扶持力度。通过财政补贴、低息贷款和"一企一策"技术支持，降低企业环保达标成本。

三是推行生态旅游承载力管理。建立景区环境监测预警系统，实施旺季预约限流、淡季营销分流，防止过度开发。

四是强化林下经济全过程监管。项目立项前必须开展环境影响评价，建立水土保持动态监测机制，确保生态效益与经济效益同步增长。

### 第二题：议论文

**平衡绿色转型的"快"与"慢"**

N市关停高耗能企业换来蓝天碧水，却面临工人转岗的阵痛；L县依托生态资源实现脱贫致富，却遭遇过度开发的隐忧。一快一慢、一东一西，折射出我国绿色发展进程中一个根本性命题：如何在生态保护与经济发展之间找到平衡点？答案或许不在非此即彼的选择中，而在于把握好"快"与"慢"的辩证法。

**绿色转型不能"一刀切"地快，需要"分类施策"的慢功夫。**N市三年关停32家高耗能企业，决心值得肯定，但"运动式"关停造成的就业冲击提醒我们，产业转型需要节奏感。对于技术落后、污染严重且无改造价值的企业，应果断关停；对于基础较好、具备改造潜力的企业，则要给时间、给政策、给技术，让它们在"慢改造"中实现"快升级"。浙江安吉的竹产业转型就是范例——不是简单关停，而是通过技术改造将传统竹加工升级为竹纤维、竹炭等绿色产品，实现了产业升级与就业稳定的双赢。

**生态经济不能"竭泽而渔"地快，需要"休养生息"的慢智慧。**L县生态旅游年接待200万人次的成绩固然亮眼，但超出承载力的开发无异于杀鸡取卵。绿色发展的核心要义是"可持续"，而非"高增长"。青海湖景区近年来实施"轮牧式"开放，每年设定游客上限并定期关闭部分区域进行生态修复，虽然短期损失了门票收入，却换来了湖滨生态系统的持续改善。这种"慢下来"的勇气，恰恰是对子孙后代最大的"快"。

**制度供给不能"等靠要"地慢，需要"先行先试"的快节奏。**企业和地方在转型中面临的困境，很大程度上源于制度供给的滞后。N市中小企业"技改投入跟不上"，L县林下经济"缺乏科学规划"，归根结底是标准体系、评估机制和扶持政策不够完善。应加快建立分行业、分区域的绿色标准体系，让企业有章可循；加快完善生态产品价值实现机制，让"绿水青山"真正可量化、可交易、可变现；加快出台绿色转型的财税金融配套政策，让先行者不吃亏、跟跑者不观望。

"快"与"慢"的辩证智慧，本质上是对发展规律的尊重。绿色发展不是一场冲刺赛，而是一场接力跑。在这场跑道上，政府需要展现"功成不必在我"的定力，企业需要拿出"十年磨一剑"的耐心，社会需要涵养"前人栽树后人乘凉"的胸怀。唯有如此，才能让蓝天碧水与百姓饭碗共存，让生态之美与发展之利同辉。

当最后一个高炉熄灭时，不是为了告别繁荣，而是为了迎来新生。这份新生，既需要壮士断腕的"快"，更需要久久为功的"慢"。
